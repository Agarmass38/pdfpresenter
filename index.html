Comment ajouter des PDFs sur GitHub

  1. Placez vos fichiers PDF dans le dossier pdfs/ de votre repository GitHub
  2. Modifiez le fichier pdfs.json pour ajouter les informations de vos PDFs
  3. Commitez et pushez vos modifications
  4. Les nouveaux PDFs apparaîtront automatiquement sur cette page
